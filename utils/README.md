# Utils

Common utilities to avoid copy-paste:
- `data.py`: dataset loading, tokenization, streaming
- `train.py`: training loop with checkpointing & logging
- `metrics.py`: BLEU/ROUGE/Perplexity/BERTScore wrappers
- `viz.py`: attention maps, confusion matrices, calibration plots
- `config.py`: experiment configuration helpers (YAML/argparse)
